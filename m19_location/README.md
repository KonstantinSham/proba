# Приложениe достопримечательности

## Основной функционал
- Отображения Яндекс карт.
- Кнопки приближения и отдаления.
- Кнопка с текущей геолокацией.
- По тапу на карте всплывает диалоговое окно с адресом и геолокацией места.
